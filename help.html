Victoria Nurse — Help

Welcome! This app is designed for home and clinic nurses to record vital signs quickly and share them with families in seconds.

How to Use

  • On the landing page, tap Get Started.
  • Fill in the patient’s vitals: temperature, pulse, respiratory rate, BP, oxygen saturation, etc.
  • Enter any Notes (symptoms, observations, instructions).
  • When finished, you can:
    • Preview & Generate PDF → saves a structured report you can download.
    • Share via WhatsApp → sends details to family/contacts.
    • Share via Email → attaches a PDF for records.

Features

  • Works fully offline — data is processed on-device.
  • No patient DOB stored — privacy-first design.
  • Optional local memory for nurse name and recent patients.
  • Multiple WhatsApp/email targets supported.
  • Speech-to-text dictation for faster notes (browser-dependent).

Troubleshooting

  • If you don’t see updates, refresh at /index.html?reset=1 to clear old caches.
  • On iOS Safari, install to Home Screen for the best fullscreen app experience.
  • Make sure microphone permissions are enabled if dictation isn’t working.
